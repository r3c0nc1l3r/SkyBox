---
title: Creating a New Project
description: How to scaffold a new matchbox-cf-worker application.
---

```bash
# 1. Create example directory
mkdir -p examples/myapp
cd examples/myapp

# 2. Write your listener
cat > MyListener.bx << 'EOF'
class MyListener {
    function onConnect(required channel) {
        channel.sendJson({"type":"welcome","service":"My App"});
    }
    function onMessage(required message, required channel) {
        channel.sendMessage("You said: " & message);
    }
    function onClose(required channel) {}
}
EOF

# 3. Copy shell files
cp ../../shell/mcf-worker.js ./
cp ../../shell/wrangler.toml ./

# 4. Create package.json
cat > package.json << 'EOF'
{
  "scripts": {
    "build": "bash ../build.sh . MyListener.bx MyListener",
    "test": "npx workerd serve test_moonphase.capnp",
    "dev": "npx wrangler dev --port 8787 --local",
    "deploy": "npx wrangler deploy"
  }
}
EOF

# 5. Build
npm run build

# 6. Dev
npm run dev
```

## Optional: Adding a Web UI

If your app serves an HTML page via `onHttpGet`, create static assets for CSS and JS:

```bash
mkdir -p assets/css assets/js

# Create your CSS file
cat > assets/css/style.css << 'EOF'
/* Your styles here */
body { font-family: sans-serif; }
EOF

# Create your JS file
cat > assets/js/app.js << 'EOF'
// Your client-side code
console.log("App loaded");
EOF
```

Then update `wrangler.toml` to serve them at the edge:

```toml
[assets]
directory = "dist/assets"
```

And add the asset copy step to your build script or create a `build-multi.sh`:

```bash
# Copy assets before build
mkdir -p dist/assets/css dist/assets/js
cp -rf assets/css/* dist/assets/css/
cp -rf assets/js/*  dist/assets/js/

# Then run standard build
bash ../build.sh . MyListener.bx MyListener
```

Reference external assets from your BoxLang HTML template:

```boxlang
private function renderPage() {
    var html = '<!DOCTYPE html><html><head>';
    html = html & '<link rel="stylesheet" href="/css/style.css">';
    html = html & '</head><body>';
    html = html & '<div id="app"></div>';
    html = html & '<script src="/js/app.js"></script>';
    html = html & '</body></html>';
    return html;
}
```

## 7. Deploy

```bash
# Build the WASM
npm run build

# Upload version (wrangler v4+)
npx wrangler versions upload

# Switch traffic
npx wrangler versions deploy --version-id <id> --percentage 100

# Or use the shortcut (wrangler < v4):
npx wrangler deploy
```

## Required Files

| File | Purpose |
|------|---------|
| `MyListener.bx` | Your BoxLang WebSocket listener class |
| `mcf-worker.js` | Worker + DO entry point (copy from `shell/`) |
| `wrangler.toml` | Cloudflare configuration (no `[wasm_modules]`!) |
| `assets/` | Optional: static files served at edge via `[assets]` |
| `dist/worker.wasm` | Compiled output (generated by build) |
