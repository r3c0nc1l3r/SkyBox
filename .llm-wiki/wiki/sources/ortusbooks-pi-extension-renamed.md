# ortus-docs extension renamed to ortusbooks-pi
Renamed the [[entities/ortus-solutions]] docs pi extension from `ortus-docs` to `ortusbooks-pi`. The extension lives at `/home/k/Git/SkyBox/.pi/extensions/ortusbooks-pi/` and provides MCP proxy tools (`search_ortus_docs`, `read_ortus_doc`) and subagents (`ortus-scholar`, `ortus-scout`) for the Ortus ecosystem documentation. All references in `settings.json`, `package.json`, `index.ts`, `PLAN.md`, and `specs/` were updated. The slash command changed from `/ortus-docs:refresh` to `/ortusbooks-pi:refresh`.
