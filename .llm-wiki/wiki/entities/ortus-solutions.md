# Ortus Solutions

Ortus Solutions is the company behind the entire Ortus ecosystem: [[entities/boxlang]], [[entities/coldbox]], [[entities/wirebox]], [[entities/cachebox]], [[entities/logbox]], [[entities/testbox]], [[entities/commandbox]], and many other products.

They maintain the official documentation on `*.ortusbooks.com` with GitBook MCP endpoints for AI agent access, which the `ortus-docs` extension uses for searching and reading documentation.

## Key Products

- **BoxLang** — Modern JVM language (successor to CFML)
- **ColdBox** — HMVC web framework
- **WireBox** — DI/AOP container
- **CommandBox** — CLI and package manager
- **TestBox** — Testing framework
- **CacheBox** — Caching framework
- **LogBox** — Logging framework
- **Quick** — ORM for BoxLang/CFML
- **qb** — Query builder
